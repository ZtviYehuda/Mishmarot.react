# הוראות הרצת הפרויקט

## בעיות שתוקנו:
1. ✅ תוקן פורט מסד הנתונים ב-.env (5435 במקום 5432)
2. ✅ הוסרו קבצי Node.js מהבקאנד (package.json, package-lock.json)
3. ✅ תוקנו הכפתורים במובייל - עכשיו מוצגים בפריסה אנכית
4. ✅ נוסף קובץ בדיקת חיבור למסד הנתונים

## שלבי הרצה:

### 1. הרצת מסד הנתונים
```bash
cd backend
docker-compose up -d
```

### 2. בדיקת חיבור למסד הנתונים (אופציונלי)
```bash
cd backend
python test_db.py
```

### 3. הרצת הבקאנד (Python Flask)
```bash
cd backend
# התקנת תלויות (פעם ראשונה)
pip install -r requirements.txt
# הרצת השרת
python run.py
```

### 4. הרצת הפרונטאנד (React + Vite)
```bash
cd frontend
# התקנת תלויות (פעם ראשונה)
npm install
# הרצת השרת
npm run dev
```

## כתובות:
- פרונטאנד: http://localhost:5173
- בקאנד: http://localhost:5000
- מסד נתונים: localhost:5435
- pgAdmin: http://localhost:8080 (admin@admin.com / admin)

## פתרון בעיות נפוצות:

### אם מסד הנתונים לא עולה:
```bash
docker-compose down
docker-compose up -d --force-recreate
```

### אם הבקאנד לא מתחבר למסד הנתונים:
1. ודא שהדוקר רץ: `docker ps`
2. בדוק את הפורט: `netstat -an | findstr 5435`
3. הרץ בדיקת חיבור: `python test_db.py`

### אם הפרונטאנד לא מתחבר לבקאנד:
1. ודא שהבקאנד רץ על פורט 5000
2. בדוק CORS בקונסול הדפדפן
3. נסה לגשת ישירות ל: http://localhost:5000/api
