# מסמך איפיון פרויקט - מערכת ניהול משמרות (CFMS)

## 1. סקירה כללית (Vision)
מערכת ה-CFMS (Cyber Force Management System) נועדה לשמש ככלי מרכזי לניהול כוח אדם, נוכחות ומעקב היררכי ביחידות טכנולוגיות/מבצעיות. המערכת מחליפה ניהול ידני או מבוסס אקסלים בממשק דינמי, מהיר ומאובטח המאפשר למפקדים לקבל תמונת מצב בזמן אמת על הפקודים שלהם.

## 2. בעלי תפקידים והרשאות
המערכת מבוססת היררכיה פיקודית, כאשר כל משתמש רואה מידע לפי ה"סקופ" (Scope) שלו:
*   **מנהל מערכת (Admin):** שליטה מלאה בהגדרות, ניהול משתמשים, איפוס סיסמאות וגיבויים.
*   **מפקד מחלקה (רע"ן/רע"ש):** צפייה בנתונים סטטיסטיים של כלל המחלקה ואישור מעברים בין מדורים.
*   **מפקד מדור (רמ"ד):** ניהול כוח האדם במדור שלו וצפייה בחתכי נוכחות מדוריים.
*   **מפקד צוות (ר"צ):** עדכון סטטוס יומי לחיילים, צפייה בלוח השנה הצוותי וניהול שוטף.

## 3. יכולות ליבה (Features)
### ניהול נוכחות (Attendance)
*   עדכון סטטוס מהיר (נוכח, חופש, מחלה, הפנייה וכו').
*   צפייה בהיסטוריית סטטוסים לכל עובד.
*   לוח שנה המציג את תמונת המצב החודשית.

### ניהול עובדים (Employee Management)
*   כרטיס עובד מפורט הכולל: פרטים אישיים, שיוך ארגוני, ורמת סיווג.
*   מנגנון חיפוש וסינון מתקדם לפי שדות מרובים.
*   ניהול מפקדים ושיוך צוותים/מדורים/מחלקות.

### מעברים ובקשות (Transfers)
*   מנגנון להגשת בקשת מעבר בין תתי-יחידות.
*   תהליך אישור דו-שלבי (מפקד יוצא ומפקד נכנס).

### דאשבורד וניתוח נתונים
*   גרפים וטבלאות המציגים חתכי נוכחות בזמן אמת.
*   התראות על ימי הולדת בצוות.
*   ייצוא נתונים לאקסל לטובת דוחות תקופתיים.

## 4. מפרט טכני (Tech Stack)
*   **Frontend:** React 18, TypeScript, Vite.
*   **Styling:** TailwindCSS, Vanilla CSS (Premium Design).
*   **Backend:** Python Flask API.
*   **Database:** PostgreSQL (תוך שימוש ב-Docker).
*   **Authentication:** JWT (JSON Web Tokens).
*   **Backup:** מערכת גיבויים אוטומטית לקבצי JSON.

## 5. מבנה הנתונים (Hierarchy)
המערכת תומכת במבנה תלת-שכבתי:
1.  **מחלקה (Department)**
2.  **מדור (Section)**
3.  **צוות (Team)**

כל עובד משויך לצוות, וכל צוות משויך למדור ולמחלקה. המפקדים מוגדרים ברמת היישות אותה הם מנהלים.
