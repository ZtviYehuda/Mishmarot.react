# מדריך הרצת Frontend

## דרישות מקדימות
- **Node.js** (גרסה 18 ומעלה)
- **npm** (מנהל החבילות שמגיע עם Node)

## שלבי ההתקנה וההרצה

### 1. כניסה לתיקייה
ודא שאתה נמצא בתיקיית `frontend`:
```bash
cd frontend
```

### 2. התקנת חבילות
לפני ההרצה הראשונה, יש להתקין את הספריות הנדרשות:
```bash
npm install
```

### 3. הרצת שרת הפיתוח
להפעלת האפליקציה במצב פיתוח:
```bash
npm run dev
```

האפליקציה תהיה זמינה בכתובת: `http://localhost:5173`

---

## פתרון בעיות נפוצות

### שגיאות חיבור לשרת (Connection Refused)
אם אתה רואה שגיאות API בקונסול, ודא ש:
1. שרת ה-Backend רץ (בפורט 5000).
2. הכתובת ב-`api.client.ts` או במשתני הסביבה תואמת לכתובת השרת.
